When the Application begins:
-user is presented with three spinners with route spinner populated with the available routes.
-user can select from the listed directions and stops
-If there is a bus service available, the app will show the list of buses and their predicted arrival timmings at the stop
-else it will display No Service message
-User can also choose to see the bus route or navigate to the stop selected from current location.


Note: Internet connection should be connected else the Progress Dialog will not dismiss and the spinners will not be populated
